# 《海阔天空》小说大纲

## 基本信息
- **男主**：海阔天空（海哥），24岁下海，表面村委干部，实际黑帮大佬
- **男二**：杰哥，海哥最好的朋友，左膀右臂
- **城市**：锦官城（成都代称）
- **产业**：按摩服务业
- **态度**：护短（核心人生信条）
- **结局**：HE，全收5女主
- **字数**：10万字
- **主线**：人生态度线 + 黑道路线双线交织

---

## 核心人物

### 男主：海阔天空（海哥）
- **年龄**：24岁下海
- **身份**：明面村委干部，暗地黑道大佬
- **产业**：按摩SPA馆
- **性格**：护短，笑面虎，该狠时绝不手软
- **信条**："谁敢动我的人，我就让他在锦官城混不下去"
- **特点**：表面和气，实则狠辣；重用情义笼络人心

### 男二：杰哥
- **全名**：待补充（可叫"陈杰"或"张杰"）
- **定位**：海哥最好的朋友，左膀右臂，参谋，百宝箱
- **年龄**：和海哥相仿
- **性格**：忠心耿耿，稳重可靠，外表粗犷实则心细如发
- **参谋能力**：
  - 每次海哥做重大决定前，杰哥都能给出关键分析
  - 擅长揣摩人心，能一眼看穿敌人弱点
  - 情报网络广，锦官城风吹草动第一时间知道
  - 关键时刻能替海哥拍板决定（海哥不在时）
- **百宝箱能力**：
  - 认识锦官城三教九流黑白两道的人
  - 各种关系门路都能搭上
  - 危急时刻总能变出意想不到的底牌
  - 有人脉、有资源、有手段
- **角色**：海哥的军师+打手+管家，日常帮海哥打理生意，关键时刻提供情报和策略，危急时刻替海哥挡刀

## 三条主线交织

### 人生态度线：护短
- **信条**："谁敢动我的人，我就让他在锦官城混不下去"
- **体现**：
  - 女员工被欺负 → 海哥出手
  - 女主们被骚扰 → 海哥周旋/反击
  - 杰哥被打 → 海哥发疯式报复
  - 敌人越嚣张 → 海哥越要护到底

### 黑道路线
- **起源**：24岁下海，从一个小按摩店做到锦官城最大的休闲产业
- **搭档**：杰哥全程辅助，帮海哥处理事务
- **身份**：明面是村委干部，负责调解纠纷
- **产业**：按摩SPA馆，技师为主
- **地位**：道上敬三分，商场上左右逢源

### 感情线
- 5个女主都是技师，在海哥手下工作
- 日久生情，各有各的动心时刻
- 全员HE，女主们从互相试探到接受彼此

## 核心爽点设计

1. **护短爽**：女主被欺负 → 海哥霸气护人 → 打脸敌人
2. **兄弟爽**：杰哥被打 → 海哥发疯式报复 → 兄弟情深
3. **参谋爽**：海哥遇到难题 → 杰哥一针见血分析 → 豁然开朗
4. **百宝箱爽**：危急时刻 → 杰哥变出意想不到的底牌 → 绝处逢生
5. **周旋爽**：表面不动声色，暗中布局让敌人自己作死
6. **修罗场爽**：五女同框各种暗流涌动，吃醋拌嘴
7. **反杀爽**：敌人嚣张 → 海哥隐忍 → 绝地反杀
8. **甜蜜爽**：五女主各自与海哥的专属甜蜜时刻

## 开篇第一章建议

**标题**：凌晨三点，海哥与杰哥

**开头节奏**：
1. 凌晨三点，海哥和杰哥在店里巡视
2. 海哥："杰哥，你觉得这店怎么样？"
3. 杰哥三言两语分析得头头是道（参谋能力初显）
4. 一个女主（建议沈幼薇）还在加班，被醉汉骚扰
5. 海哥三言两语震慑醉汉，护住女主
6. 女主震惊：原来老板这么狠
7. 海哥淡淡一笑："我的人，谁都不能动"
8. 杰哥在旁边默默点头，事后低声告诉海哥："哥，这醉汉背后有人，我查查。"（百宝箱初显）
9. 引出背景：24岁下海，第一家店，杰哥是铁兄弟+军师+百宝箱

## 备注

- 无警察线，纯黑道商战感情
- 杰哥定位：海哥的影子，但每次出现都高光。参谋+百宝箱是两大标签
- 海哥负责决策和护短，杰哥负责情报、分析、人脉、执行
- 尺度可参考都市爽文，暧昧合理，脖子以下不写
- 节奏：前5章快速铺人物，第6章完成团队集结
- 矛盾升级要有层次：第1个敌人 → 2个 → 3个联手 → 最终决战
