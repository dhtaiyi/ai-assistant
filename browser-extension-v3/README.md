# OpenClaw 浏览器控制扩展

**纯浏览器扩展，完整控制浏览器 - 无需Python、无需服务器！**

## 功能

- 🌐 打开任意网页
- 👆 点击页面元素
- ⌨️ 输入文本
- 📜 滚动页面
- 📊 获取同花顺股票数据
- 💻 执行JavaScript代码
- 🔍 查找页面元素

## 安装方法

1. 打开Chrome，地址栏输入：`chrome://extensions/`
2. 打开右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `browser-extension-v3/` 文件夹

## 使用方法

1. 点击工具栏上的扩展图标
2. 在弹出窗口中操作：
   - **导航**：输入URL，点击「跳转」
   - **点击**：输入CSS选择器，点击「点击」
   - **数据**：点击「获取同花顺数据」

## 常见问题

Q: 扩展不工作？
A: 刷新页面重试

Q: 点击失败？
A: 检查CSS选择器是否正确

Q: 获取不到数据？
A: 页面可能未完全加载，等待几秒再试
