由于服务器网络限制，建议在本地运行以下程序获取实时数据：

## 使用方法

### 方法1：双击运行
```
双击 stock_plate_simple.bat
```

### 方法2：命令行运行
```
python stock_plate.py
```

## 数据来源

使用 **新浪股票API** 获取实时数据：
- URL: `http://hq.sinajs.cn/list=sh600519`
- 格式: 股票名称,开盘价,昨收价,当前价,最高价,最低价...

## 文件说明

- `stock_plate.py` - Python完整版（需要Python环境）
- `stock_plate_simple.bat` - Windows批处理版
- `stock_plate.bat` - 简化批处理版
