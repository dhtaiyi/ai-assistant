# OpenClaw Browser Control (纯扩展版)

**纯Chrome扩展，无需Python、无需服务器，直接控制浏览器！**

## 安装方法

1. 打开Chrome，地址栏输入：`chrome://extensions/`
2. 打开右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `browser-control-pure/` 文件夹

## 使用方法

### 1. 点击扩展图标

安装完成后，点击工具栏上的扩展图标，打开控制面板。

### 2. 导航到网页

在输入框输入URL，点击「跳转」。

### 3. 控制浏览器

| 功能 | 操作 |
|------|------|
| 点击元素 | 输入CSS选择器，点击「点击」 |
| 输入文本 | 输入选择器和文本，点击「输入」 |
| 滚动页面 | 点击方向按钮 |
| 获取数据 | 点击「获取股票数据」 |
| 执行代码 | 输入JavaScript代码，点击「执行」 |

## 快捷链接

点击即可快速打开：
- 同花顺
- 东方财富
- 雪球
- 百度

## 获取同花顺数据

1. 点击扩展图标
2. 点击「获取股票数据」
3. 查看结果（显示价格、涨跌）

## 常见问题

Q: 扩展不工作？
A: 检查是否已启用，刷新页面重试

Q: 点击失败？
A: 检查CSS选择器是否正确

Q: 页面加载慢？
A: 等待页面完全加载后再操作

## 文件结构

```
browser-control-pure/
├── manifest.json    # 扩展配置
├── background.js   # 后台服务
├── popup.html      # 控制界面
├── popup.js        # 界面逻辑
└── README.md       # 说明文档
```

## 权限说明

- `activeTab`: 访问当前标签页
- `scripting`: 执行JavaScript
- `tabs`: 管理标签页
- `host_permissions`: 访问所有网站
