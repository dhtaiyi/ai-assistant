# 自主学习技能 (self-learning)

## 功能

持续学习和优化的自动化系统。

## 核心功能

### 1. 持续学习
- 自动搜索和安装新技能
- 根据需求推荐技能
- 学习时长可配置（默认6小时）

### 2. 知识积累
- 记录学习笔记
- 保存有用的信息
- 构建个人知识库

### 3. 自我优化
- 分析使用模式
- 优化技能配置
- 改进响应效果

## 使用方法

### 基本用法
```bash
# 学习6小时（默认）
python self_learning.py

# 学习指定时长
python self_learning.py --hours 4

# 学习8小时
python self_learning.py --hours 8
```

### 自定义配置
```bash
# 禁用自动安装
python self_learning.py --install false

# 禁用技能优化
python self_learning.py --optimize false
```

## 配置选项

| 参数 | 默认值 | 说明 |
|------|--------|------|
| --hours | 6 | 学习时长（小时） |
| --install | true | 自动安装技能 |
| --optimize | true | 优化技能 |

## 注意事项

1. **网络要求** - 学习过程中需要网络连接
2. **保持运行** - 不要中断脚本运行
3. **磁盘空间** - 知识库会占用一定空间
4. **学习频率** - 建议每周运行1-2次
