# 自主学习系统

## 功能

1. **持续学习** - 运行时长可配置（默认6小时）
2. **技能更新** - 自动搜索和安装新技能
3. **知识积累** - 记录学习内容和成果
4. **自我优化** - 根据使用情况优化技能

## 核心模块

### 1. 技能学习器
- 自动搜索 ClawHub 新技能
- 根据需求推荐技能
- 安装和配置新技能

### 2. 知识积累器
- 记录学习笔记
- 保存有用的信息
- 构建知识库

### 3. 自我优化器
- 分析使用模式
- 优化技能配置
- 改进响应效果

## 使用方法

```bash
python self_learning.py --hours 6
```

## 配置文件

```json
{
  "learning_hours": 6,
  "auto_install": true,
  "save_knowledge": true,
  "optimize_skills": true
}
```

## 学习目标

- 掌握新技能
- 积累知识
- 优化系统
- 提升能力

## 注意事项

1. 学习过程中保持网络连接
2. 定期保存学习成果
3. 避免过度学习
4. 注意休息
