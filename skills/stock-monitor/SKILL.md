# 股票盯盘 Skill

实时监控 A股/港股/美股 + 智能板块推荐。

## 支持的股票

| 市场 | 示例代码 | 说明 |
|------|----------|------|
| A股 | 600519 | 贵州茅台 |
| A股 | 000001 | 平安银行 |
| 港股 | 00700 | 腾讯控股 |
| 美股 | AAPL | 苹果 |

## 使用方式

### 基础功能

```bash
# 查询实时价格
stock price 600519

# 监控股票
stock watch 600519 2000    # 设置目标价
stock watch 00700 400     # 港股

# 查看监控列表
stock list

# 实时监控 (10次)
stock realtime AAPL
```

### 板块分析

```bash
# 行业板块
stock industry

# 概念板块
stock concept

# 今日热点
stock hot
```

### 智能推荐

```bash
# 按行业推荐
stock recommend 半导体
stock recommend AI人工智能
stock recommend 新能源
stock recommend 医药
stock recommend 白酒

# 支持的行业:
# 半导体, AI人工智能, 光伏, 锂电, 医药, 白酒
# 银行, 证券, 房地产, 消费, 军工, 5G, 汽车
# 互联网, 新能源车, 储能, CRO, 食品饮料, 家电
```
