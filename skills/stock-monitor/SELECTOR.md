# 高级选股 Skill

涨速榜 + 结构突破 + 强势股选股。

## 使用方式

```bash
# 涨速榜
stock-selector up-speed          # 涨速 > 1% 的板块

# 结构突破
stock-selector breakthrough     # 平台突破/箱体突破

# 强势股
stock-selector strong-list      # 3日涨幅 > 5%

# 成交量异动
stock-selector volume-break     # 量比 > 2

# 金叉选股
stock-selector cross            # MA5 > MA10 > MA20

# 创新高
stock-selector high-break       # 突破历史高点

# 综合选股
stock-selector all             # 涨速榜 + 结构突破 (推荐)
```

## 命令对照

| 命令 | 说明 |
|------|------|
| `up-speed` | 涨速榜 (涨速 > 1%) |
| `breakthrough` | 结构突破选股 |
| `strong-list` | 强势股排行 |
| `volume-break` | 量能异动 |
| `cross` | 金叉选股 |
| `high-break` | 创新高选股 |
| `all` | 综合选股 |

## 选股逻辑

### 涨速榜
- 板块涨速 > 1%
- 识别资金快速流入的板块

### 结构突破
- 股价突破近期平台/箱体
- 均线多头排列
- MACD 金叉

### 综合选股
```
✓ 板块涨速 > 1%
✓ 个股涨速 > 2%
✓ 股价突破平台
✓ 量比 > 1.5
✓ 均线金叉
```

## 评分标准

| 评分 | 说明 |
|------|------|
| ★★★★★ 强烈关注 | 涨速快 + 形态突破 + 量能配合 |
| ★★★★☆ 关注 | 涨速较好 + 形态良好 |
| ★★★☆☆ 观察 | 涨速一般，趋势未明 |
