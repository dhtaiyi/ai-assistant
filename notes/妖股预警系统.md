## 四、妖股预警使用指南

### 4.1 每日操作流程

```
⏰ 09:30 开盘前
   → 运行预警扫描
   → 关注昨日涨停股今日开盘表现

⏰ 10:00-10:30 早盘
   → 观察开盘第一波走势
   → 识别放量突破的股票

⏰ 13:00-14:00 午盘
   → 二次确认信号
   → 回调到均线支撑的是买点

⏰ 14:30-15:00 尾盘
   → 做决策：买入/观望/止损
   → 记录今日预警股票池
```

### 4.2 预警信号处理

| 信号 | 应对策略 |
|------|---------|
| 💥红色预警(≥80) | 可追涨，仓位10-20%，严格止损 |
| 🔴橙色预警(65-79) | 等回调买入，仓位5-10% |
| 🟡黄色预警(50-64) | 轻仓试探，仓位<5% |
| 🔵蓝色观察(<50) | 不操作，继续观察 |

### 4.3 止损策略

```
妖股止损原则：
1. 买入后下跌 > 7% → 无条件止损
2. 跌破5日均线 → 减半仓
3. 涨停炸板且回封失败 → 卖出
4. 板块退潮 → 果断离场
```

妖股启动三大铁律：

1️⃣ 量比 > 1.5x 是必要条件

2️⃣ 振幅扩大是加速信号
   → 振幅突然放大2倍以上 = 主升浪确认

3️⃣ 最佳买入点在首次涨停后
   → 首板次日高开回调是黄金买点
   → 追二板风险急剧增加

## 六、预警系统局限性与风控

### 6.1 系统局限

⚠️ **不能预测题材**：系统只能识别形态，无法预判题材持续性
⚠️ **不能预测高度**：预警只显示概率，不预测涨幅
⚠️ **假信号**：放量可能只是出货，不是启动

### 6.2 风控原则

```
1. 单只仓位 ≤ 20%
2. 总妖股仓位 ≤ 50%
3. 亏损 -7% 必须止损
4. 连续2次止损 → 暂停操作1天
5. 市场情绪极差时 → 空仓
```
