## 一、妖股的定义

**妖股**：短期内（通常5-20个交易日）股价涨幅超过100%的个股，伴随以下特征：
- 连续涨停或间歇性涨停
- 换手率异常高
- 市场关注度极高
- 常有题材/概念加持

## 二、妖股核心特征（待填充研究数据）

### 2.1 基本面特征
| 指标 | 妖股常见特征 |
|------|-------------|
| 总市值 | 通常 < 100亿（便于资金拉升） |
| 流通市值 | 通常 < 50亿 |
| 股东户数 | 集中度较高 |
| 资产负债率 | 无特定规律 |
| 行业属性 | 热门赛道/题材 |

### 2.2 技术面特征
| 指标 | 特征描述 |
|------|---------|
| 均线形态 | 蓄势整理后放量突破 |
| K线形态 | 长阳、长下影线、跳空缺口 |
| 成交量 | 温和放量 → 突然爆发 |
| 换手率 | 启动时5%-15%，高位 > 20% |
| MACD | 零轴上方金叉或水下金叉 |
| RSI | 启动时 < 70，高位 > 85 |

### 2.3 资金面特征
- 主力资金净流入持续多日
- 龙虎榜常有名营业部现身
- 融资融券余额变化

### 2.4 题材/概念特征
- 站上市场热点（新能源、AI、消费等）
- 政策利好驱动
- 行业周期性爆发
- 重组/并购预期

## 三、妖股筛选指标体系

### 筛选维度（待量化）
1. **价格动量**：5日涨幅 > 20%，10日涨幅 > 40%
2. **成交量突变**：今日成交量 > 昨日2倍
3. **市值适中**：流通市值10-80亿
4. **换手率**：5% < 换手率 < 25%
5. **均线多头**：MA5 > MA10 > MA20
6. **突破关键位**：突破前期高点/平台

### 加分项
- 近一个月有涨停记录
- 所属板块今日涨幅前5
- 有明确利好消息
- 实际控制人变更/重组预期

### 减分项
- 连续高位横盘
- 大股东拟减持
- 监管问询函
- 解禁压力

## 七、待深入研究

### 7.1 妖股生命周期
1. **潜伏期**：缩量整理，均线粘合
2. **启动期**：放量突破，热点出现
3. **主升期**：连续拉升，涨停不断
4. **高位震荡**：换手率急剧放大
5. **出货期**：高位长阴，利好兑现
6. **退潮期**：绵绵阴跌

### 7.2 关键时间节点
- 首板：启动信号
- 3板：成妖确认
- 5板：妖性大发
- 7板以上：高度妖股，风险加剧
