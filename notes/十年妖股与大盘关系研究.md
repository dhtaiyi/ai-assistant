## 三、妖股与大盘关系规律

### 3.1 妖股数量与市场周期

```
市场周期与妖股关系：

📈 牛市初期（2014-2015, 2019）
   → 妖股数量：较少（资金观望）
   → 特点：龙头先涨，题材扩散

📈 牛市中期（2015, 2020）
   → 妖股数量：爆发式增长
   → 特点：全面开花，垃圾股也能飞

📈 牛市末期（2015中, 2021）
   → 妖股数量：见顶
   → 特点：妖股开始分化，真正龙头确立

📉 熊市（2018, 2022）
   → 妖股数量：急剧减少
   → 特点：极少数妖股逆势（避险+题材）

📊 震荡市
   → 妖股数量：中等
   → 特点：结构性行情，板块轮动妖
```

### 3.2 妖股与大盘指数关系

| 大盘状态 | 妖股特征 | 操作建议 |
|---------|---------|---------|
| **大盘上涨趋势** | 妖股数量多、涨幅大 | 积极参与 |
| **大盘横盘震荡** | 结构性妖股、轮动快 | 快进快出 |
| **大盘下跌趋势** | 妖股少、风险高 | 观望为主 |
| **大盘底部区域** | 妖股开始出现 | 布局优质标的 |

## 六、妖股筛选情绪指标体系

### 6.1 情绪判断标准

| 情绪阶段 | 涨停家数 | 炸板率 | 操作策略 |
|---------|---------|--------|---------|
| **冰点** | <10家 | >50% | 空仓等待 |
| **低迷** | 10-30家 | 30-50% | 轻仓试探 |
| **回暖** | 30-50家 | 20-30% | 积极参与 |
| **高潮** | >50家 | <20% | 重仓龙头 |
| **退潮** | 快速减少 | >40% | 果断离场 |

### 6.2 妖股最佳买入时机

```
1. 【大盘共振】大盘企稳 + 妖股启动日
2. 【情绪转折】昨日涨停今日高开反包
3. 【缩量回调】涨停后缩量回调到5日线
4. 【板块启动】所属板块首次高潮日
5. 【空间板突破】连板高度突破近期天花板
```

## 八、核心结论

### 8.1 妖股与大盘关系

1. **妖股是大盘的先行指标**：妖股往往先于大盘见顶/见底
2. **数量与情绪正相关**：涨停家数是市场情绪的晴雨表
3. **熊市妖股更珍贵**：熊市中能逆势上涨的妖股更具研究价值
4. **板块轮动规律**：每轮牛市1-2个主线板块，妖股集中于此

### 8.2 当前市场判断

- **大盘状态**：弱势下跌（2026-04-03）
- **妖股数量**：中等（38+只涨停）
- **市场情绪**：谨慎/试探期
- **建议策略**：轻仓参与首板，不追高位股

### 8.3 妖股筛选核心条件

```
✅ 流通市值10-50亿
✅ 今日涨幅5-15%（未到顶）
✅ 换手率8-25%（资金活跃）
✅ 量比>2（增量资金入场）
✅ 首板优先（2板以上风险递增）
✅ 大盘非暴跌日
```
