```

2. 同价格下，时间优先
3. 最高买入价 = 最低卖出价时成交
```

## 三、股票异动监管规则

### 3.1 异常波动标准

根据《上海/深圳证券交易所交易规则》：

#### 深圳证券交易所

| 情形 | 触发条件 |
|------|---------|
| **连续3个交易日内日收盘价涨跌幅偏离值** | 累计达到±20% |
| **连续3个交易日内日均换手率** | 与基准指数偏离值累计达到±30% |
| **连续3个交易日内日收盘价涨跌幅偏离值** | 达到±7%且基准指数涨跌幅达到±5% |
| **证券价格** | 达到全天最高/最低成交价±20% |

#### 上海证券交易所

| 情形 | 触发条件 |
|------|---------|
| **无价格涨跌幅限制的证券** | 当日涨跌幅10%以上，或市价1元以上且涨跌幅5%以上 |
| **有价格涨跌幅限制的A股、B股** | 连续3个交易日内日收盘价格涨跌幅累计达到±20% |
| **封闭式基金** | 连续3个交易日内日收盘价格涨跌幅累计达到±20% |
| **严重异常波动** | 连续10个交易日内3次出现同向异常波动 |

### 3.2 异动公告标准

上市公司应发布异动公告的条件：

```
触发条件（通常为3个交易日）：
1. 累计涨跌幅偏离值达到±20%（创业板/科创板±30%）
2. 连续3个交易日触及涨跌幅限制
3. 换手率达到20%以上
4. 交易所认定的其他情形
```

### 3.3 异动期间监管措施

| 阶段 | 措施 |
|------|------|
| **一般异动** | 要求上市公司核查并公告 |
| **严重异动** | 停牌核查（通常1个交易日） |
| **异常波动** | 可能被纳入重点监控 |

### 3.4 妖股常见监管红线

```
🚨 妖股异动监管红线：

1. 【涨幅过快】3日偏离值≥20%
   → 发异动公告
   → 可能被特停

2. 【换手率过高】换手率≥20%
   → 交易所重点关注
   → 主力可能被抓

3. 【连续涨停】连续3个涨停
   → 强制停牌核查
   → 典型案例：10连板以上

4. 【严重异常波动】
   → 10个交易日内3次异动
   → 可能被重点监控

5. 【信息披露违规】
   → 蹭热点/虚假信息
   → 证监会立案调查
```

## 四、妖股特停规则（重点）

### 4.1 特停规则一览

| 板块 | 特停标准 | 特停时长 |
|------|---------|---------|
| **主板** | 连续3-5个涨停 | 通常1天 |
| **创业板** | 3个交易日涨幅偏离值≥30% | 停牌核查 |
| **科创板** | 无涨跌幅限制，波动大 | 停牌核查 |
| **北交所** | 30%涨跌幅限制 | 较少特停 |

### 4.2 历史特停案例

| 股票 | 时间 | 连板数 | 特停时长 |
|------|------|--------|---------|
| 东方通信 | 2019年2月 | 10连板后特停 | 2个交易日 |
| 星期六 | 2019年12月 | 13连板后特停 | 3个交易日 |
| 圣龙股份 | 2023年10月 | 14连板后特停 | 5个交易日 |
| 克来机电 | 2024年2月 | 13连板后特停 | 待查 |

### 4.3 特停后的走势规律

```
特停后走势规律：

1. 【一字板开板后特停】
   → 复牌后通常还有1-2个板
   → 属于"妖性未死"

2. 【高位特停】
   → 复牌后可能直接下跌
   → 主力借机出货

3. 【特停期间板块退潮】
   → 复牌后补跌概率大
   → 应果断离场
```

## 五、实战注意事项

### 5.1 集合竞价 tricks

```
开盘前必看：

09:15-09:20  ⚠️ 可撤单，假信号多
   → 不要追此时的涨停板
   → 很多主力故意挂涨停价诱导

09:20-09:25  ✅ 真实竞价区间
   → 看这个阶段的价格变化
   → 高开幅度>5%要谨慎追入

09:25 公布开盘价
   → 如果大幅低开，可能开盘即卖
   → 如果大幅高开，获利盘可能砸盘
```

### 5.2 妖股操作时间窗口

```
最佳买卖时间点：

✅ 【09:30-09:45】早盘第一波
   → 判断当日强弱
   → 妖股通常早盘发动

✅ 【10:00-10:30】分歧点
   → 回调是买点
   → 放量突破是确认

✅ 【13:30-14:00】午盘异动
   → 部分妖股午盘发动
   → 跟随资金方向

⚠️ 【14:30以后】尾盘
   → 追涨风险大
   → 主力可能尾盘出货
```

### 5.3 避开监管的注意事项

```
避免被重点监控：

1. 不要单日换手率>50%
2. 不要连续3日换手率>30%
3. 不要蹭热点发公告
4. 不要连续涨停后发利好
5. 关注交易所问询函
```
