## 📖 核心口诀

### 选股
```
人气 > 强度 > 题材纯度 > 盘子大小
单价<10元 | 换手率>5% | 量比>2x
```

### 买点
```
竞价超预期 | 分歧转一致 | 首阴反包 | 回调5日线
```

### 卖点
```
-7%止损 | 破5日线 | 情绪退潮 | 断板走A
```

### 情绪周期
```
上升期：涨停>50家，重仓龙头
混沌期：涨停20-50家，轻仓补涨
退潮期：涨停<20家，空仓休息
```
