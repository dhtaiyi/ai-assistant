## 三、妖股评分模型（v1.0）

### 评分维度与权重

```
妖股综合评分 = Σ(各维度得分 × 权重)

1. 涨幅得分 (30%)
   - 5-10% → +30分 ✅
   - 10-20% → +25分 ⚠️（涨幅偏大）
   - 3-5% → +20分 ⚠️（起步较慢）
   - >20% → +15分 ❌（风险累积）

2. 换手率得分 (25%)
   - 8-15% → +25分 ✅（最健康）
   - 15-25% → +20分 ⚠️（偏高）
   - >25% → +10分 ❌（主力可能出货）
   - 5-8% → +15分 ⚠️（尚可）

3. 流通市值得分 (20%)
   - <30亿 → +20分 ✅（最优拉升盘）
   - 30-50亿 → +15分 ✅
   - 50-100亿 → +10分 ⚠️（偏大）
   - >100亿 → +5分 ❌（大盘股难妖）

4. 量比得分 (15%)
   - >3x → +15分 ✅
   - 2-3x → +12分 ✅
   - 1.5-2x → +8分 ⚠️
   - <1.5x → +3分 ❌

5. 市盈率得分 (10%)
   - 0-30 → +10分 ✅（估值合理）
   - 亏损 → +5分 ⚠️（风险与机会并存）
   - >50 → +3分 ❌（估值偏高）
```

## 四、龙头战法实操要点（来自课程）

### 选股维度（优先级排序）
> **人气 > 强度 > 题材纯度 > 盘子大小**

### 买点信号
1. **竞价超预期**：高开3-7%
2. **分歧转一致**：放量涨停板
3. **首阴反包**：第一根阴线后的反包板
4. **缩量回调**：缩量跌回均线支撑

### 卖点信号
1. 涨停封不住
2. 板块退潮，跟风股先行下跌
3. 跌破5日均线
4. 单日亏损-7%至-10%止损

### 仓位管理
- 只持有1-2只，集中持仓
- 永不扛单，走弱果断离场
- 情绪低迷时空仓
