## 二、选股标准（龙头战法核心）

### 2.1 选股优先级

```
选股维度（优先级从高到低）：

🎯 人气 > 强度 > 题材纯度 > 盘子大小

1️⃣ 人气（最重要）
   → 涨停时间早、封板坚决
   → 板块内涨幅最大
   → 带动同板块个股跟风

2️⃣ 强度
   → 今日涨幅/连板数
   → 突破关键位置
   → 分时图流畅度

3️⃣ 题材纯度
   → 纯正龙头（非跟风补涨）
   → 题材新颖/政策支持
   → 有实质性利好

4️⃣ 盘子大小
   → 流通市值10-50亿最佳
   → 太大了拉不动
   → 太小了流动性差
```

### 2.2 妖股筛选条件（量化版）

```
✅ 基础条件（必须满足）：
   - 流通市值 < 100亿（最佳10-50亿）
   - 今日涨幅 > 3%
   - 换手率 > 5%

✅ 加分条件（满足越多越好）：
   - 涨幅 5-15%（起步阶段）
   - 换手率 8-20%（资金活跃）
   - 量比 > 2倍（成交量放大）
   - 5日内有涨停记录
   - 所属板块涨幅 > 2%
   - 市盈率为负但非ST

❌ 减分/排除条件：
   - 涨幅 > 20%（可能已到顶）
   - 换手率 > 30%（主力可能出货）
   - ST股（风险极高）
   - 高位长阴（主力出货形态）
```

## 三、买入时机（三大买点）

### 3.1 竞价超预期买入

```
📊 竞价超预期模式：

✅ 超预期特征：
   - 昨日涨停，今日高开3-7%
   - 竞价量比昨日放大
   - 板块整体强势

⚠️ 低于预期：
   - 高开低走
   - 竞价量萎缩
   → 不买入

⏰ 操作时间：
   - 09:20-09:25确认信号
   - 09:30后确认封板
```

### 3.2 分歧转一致打板

```
📊 分歧转一致模式：

形态：
   - 上午涨停打开，震荡
   - 14:00前再次封板
   - 封板坚决，无炸板

确认信号：
   - 封板后成交量稳定
   - 同板块个股跟风
   - 大盘非暴跌日

⚠️ 风险：
   - 炸板后不回封 = 卖出信号
   - 尾盘14:30后涨停板不打
```

### 3.3 首阴反包

```
📊 首阴反包模式：

适用场景：
   - 连续涨停后，第一根阴线
   - 阴线当天成交量温和
   - 次日高开反包

买入点：
   - 次日高开高走
   - 突破阴线最高点
   → 买入

❌ 不操作：
   - 阴线放大量（主力出货）
   - 连续两根阴线
   - 大盘退潮期
```

## 四、卖出时机（三大卖点）

### 4.1 止损卖出

```
🚨 强制止损线：

1️⃣ -7%止损（铁律）
   → 买入后下跌超过7%
   → 必须无条件卖出
   → 不幻想反弹

2️⃣ 跌破5日线减半仓
   → 收盘跌破
   → 次日反抽不过均线卖出

3️⃣ 涨停炸板不回封
   → 当天尾盘卖出
   → 不等次日开盘
```

### 4.2 主动止盈

```
🎯 止盈策略：

妖股止盈（连板股）：
   - 首板次日不连板 → 卖出1/3
   - 断板（不再涨停）→ 卖出1/2
   - 跌破5日线 → 全部卖出

普通股止盈：
   - 涨幅>15%，分批卖出
   - 先卖一半，剩下博趋势
   - 趋势走坏全部卖出

补涨股止盈：
   → 补涨的定义：一波主升后的跟风股
   → 补涨到哪算哪，不恋战
```

### 4.3 卖出信号清单

```
🚨 以下情况立即卖出：

1. 涨停封不住，炸板
2. 14:30后涨停打开
3. 跌停板出现
4. 跌破关键支撑位
5. 板块龙头走弱/跌停
6. 大盘暴跌（-2%以上）
7. 突发利空公告
8. 连续2天放大量但不涨
9. 龙虎榜显示知名游资出货
10. 收到监管问询函
```

## 五、超短情绪周期判断

### 5.1 三种周期识别

```
📊 超短情绪三周期：

【上升期】✅ 最佳赚钱时机
特征：
   - 涨停家数 > 50
   - 跌停家数 < 5
   - 连板股晋级率高
   - 空间板持续拓展
策略：
   → 积极参与龙头
   → 重仓核心标的

【混沌期】⚠️ 谨慎操作
特征：
   - 涨停家数 20-50
   - 跌停家数 5-15
   - 连板高度下降
   - 高低切换频繁
策略：
   → 轻仓参与
   → 首选补涨机会

【退潮期】❌ 空仓休息
特征：
   - 涨停家数 < 20
   - 跌停家数 > 15
   - 空间板A字杀
   - 龙头断板即大跌
策略：
   → 不交易就是最好的交易
   → 空仓等待冰点
```

### 5.2 每日情绪检查清单

```
⏰ 每日必看（开盘前）：

1. 昨日涨停股今日表现
   → 首板多 = 情绪好
   → 昨日龙头低开 = 情绪差

2. 今日涨停/跌停数量
   → 涨停>50 = 上升期
   → 涨停<20 = 退潮期

3. 空间板高度
   → 7板+ = 强势
   → 3板就断 = 弱势

4. 板块联动
   → 有明确主线 = 可操作
   → 无主线无热点 = 休息

5. 大盘指数
   → 大盘站稳均线 = 可操作
   → 大盘破位 = 休息
```

## 六、K线形态实战应用

### 6.1 看涨K线形态

```
✅ 有效看涨信号：

1. 【大阳线突破】
   - 放量突破前期高点
   - 换手率适中（8-15%）
   - 配合板块共振

2. 【仙人指路】
   - 长上影线试探压力
   - 次日阳线收复
   → 突破上影线高点 = 买入

3. 【早晨之星】
   - 下跌后出现十字星
   - 第三根阳线收复
   → 阳线确认 = 买入

4. 【红三兵】
   - 三连阳稳步上涨
   - 各有上下影线
   → 缩量回调是买点

5. 【缩量涨停】
   - 涨停但换手率低
   → 主力控盘强 = 持股待涨
```

### 6.2 看跌K线形态

```
❌ 危险卖出信号：

1. 【黄昏之星】
   - 连续上涨后出现
   - 十字星/小实体
   → 第三根阴线 = 卖出

2. 【高位长阴】
   - 涨幅大+放大量
   → 主力出货 = 卖出

3. 【乌云盖顶】
   - 阳线后高开低走
   - 阴线深入阳线实体
   → 卖出

4. 【墓碑线】
   - 长上影线收盘接近最低
   → 当天尾盘卖出

5. 【吊颈线】
   - 高位锤子线
   → 卖出
```

## 七、筹码理论实战应用

### 7.1 筹码峰判断支撑压力

```
📊 筹码分布三法则：

1️⃣ 支撑判断
   - 股价回落至筹码下峰
   - 缩量企稳 = 支撑有效
   → 低吸买点

2️⃣ 压力判断
   - 股价上涨至筹码上峰
   - 放量不过 = 压力有效
   → 卖出信号

3️⃣ 突破确认
   - 放量突破筹码峰
   - 峰由压力变支撑
   → 买入确认
```

### 7.2 威科夫吸筹/派发

```
📊 威科夫五大阶段：

【吸筹区（A-E阶段）】
   A: PS/SC（初步支撑/抛售高潮）
   B: 机构建仓
   C: Spring（测试支撑）
   D: SOS（强势初现）
   E: 突破区间
   → E阶段突破 = 买入

【派发区（A-E阶段）】
   A: PSY/BC（初步供应/抢购高潮）
   B: 机构派发
   C: UT（上冲回落）
   D: SOW（弱势信号）
   E: 跌破区间
   → E阶段跌破 = 卖出
```

## 八、妖股专项策略

### 8.1 妖股生命周期

```
📊 妖股五阶段：

1️⃣ 【潜伏期】
   - 缩量整理，均线粘合
   - 不操作，观察

2️⃣ 【启动期】
   - 放量涨停突破
   - 首板 = 确认信号
   → 可追涨买入

3️⃣ 【主升期】
   - 连续涨停
   - 3板成妖，5板确认
   → 回调5日线买入

4️⃣ 【高位震荡】
   - 换手率急剧放大
   - 振幅加大
   → 逐步减仓

5️⃣ 【退潮期】
   - 高位长阴
   - 跌停出现
   → 清仓离场
```

### 8.2 妖股买入时机

```
✅ 妖股最佳买入点：

1️⃣ 首板次日高开
   - 高开3-7%
   - 快速封板
   → 确定性最高

2️⃣ 3板分歧转一致
   - 第三板炸板后回封
   → 妖性确认

3️⃣ 5日线低吸
   - 连板后回调5日线
   - 缩量企稳
   → 博反弹

4️⃣ 首阴反包
   - 第一根阴线
   - 次日阳线反包
   → 妖股2波预期
```

## 九、每日复盘清单

```
📋 每日复盘（收盘后30分钟）：

1️⃣ 市场情绪
   - 涨停/跌停数量
   - 连板高度
   - 空间板

2️⃣ 持仓股分析
   - 当日表现
   - 是否触发止损
   - 次日计划

3️⃣ 明日候选
   - 选出3-5只观察
   - 制定买入计划
   - 设定买入价格

4️⃣ 风险评估
   - 大盘环境
   - 持仓仓位
   - 是否需要减仓

5️⃣ 心态检查
   - 是否有冲动交易
   - 是否遵守计划
   - 是否有遗憾
```

## 十、风控铁律

```
🚨 十大风控铁律（违反任何一条就亏钱）：

1. -7%必须止损，不幻想
2. 不与跌停板发生关系
3. 永不加仓摊成本
4. 单票仓位≤20%
5. 总仓位根据情绪调整
6. 连续2次止损，休息1天
7. 大盘破位必须减仓
8. 不听消息炒股
9. 不买ST/亏损/问题股
10. 情绪退潮时空仓

⚠️ 记住：
   → 活着最重要
   → 亏损后急于回本 = 亏更多
   → 轻仓练手，等学成再加仓
```

## 十一、策略总结

```
📊 交易策略流程图：

开盘前：
   ├─ 复盘昨日情绪
   ├─ 选出观察标的
   └─ 制定买入计划

   ├─ 09:15-09:25 竞价判断

   ├─ 复盘今日操作

   └─ 制定明日计划

选股：
   └─ 人气>强度>题材>盘子

买点：
   └─ 竞价超预期/分歧转一致/首阴反包

卖点：
   └─ -7%止损/破均线/情绪退潮
```
