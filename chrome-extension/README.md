# 小红书助手 - Chrome 浏览器插件

## 功能特性

### 1. AI 内容创作
- 输入标题，一键生成正文
- 支持多种主题（效率、护肤、美食、旅游）
- 本地模板备用

### 2. 一键发布
- 直接发布到小红书
- 自动保存草稿
- 定时发布功能

### 3. 数据统计
- 今日浏览量
- 点赞数
- 评论数

### 4. 快捷工具
- 清除缓存
- 设置管理
- 使用帮助

## 安装方法

### 方法1：开发者模式（推荐）

1. 下载插件文件
2. 打开 Chrome 浏览器
3. 地址栏输入：`chrome://extensions/`
4. 打开「开发者模式」（右上角）
5. 点击「加载已解压的扩展程序」
6. 选择插件文件夹

### 方法2：打包安装

1. 打包插件：chrome://extensions/ → 打包扩展程序
2. 生成 .crx 文件
3. 拖拽到 Chrome 安装

## 使用方法

1. 点击工具栏图标打开插件
2. 输入标题，点击「AI帮我写」
3. 编辑内容，点击「发布到小红书」
4. 或设置定时发布

## 依赖服务

- **OpenClaw API** (可选): localhost:8080
- **小红书 MCP** (可选): localhost:18060

如果服务不可用，会使用本地模板。
